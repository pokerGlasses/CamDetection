**Usage**

Install dependencies (Must be python 3.12 or lower, I recommend 3.10)
python3.10 -m pip install inference supervision opencv-python
(Depends on python installation)

Find API Key for Roboflow:
- Create account/sign in
- Create Workspace
- Go to settings -> API key -> private API key
- Paste into api_key variable

DO NOT commit api keys. 

DO NOT overuse inferences -- I believe we only get 15 credits on free plan so need to save some for live demos and actual raspberry pi testing.
